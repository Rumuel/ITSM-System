# ITSM-System

Sistema de Gestao de Infraestruturas e Incidentes desenvolvido no contexto da UFCD 10790 - Projeto de Programacao.

## Objetivo
Desenvolver uma plataforma ITSM para gestao de ativos informaticos, registo de incidentes e atribuicao automatica de tickets ao tecnico mais adequado.

## Stack tecnologico
- ASP.NET Core Web API
- ReactJS com TypeScript
- SQL Server
- Entity Framework Core
- JWT
- Swagger/OpenAPI

## Estrutura
- backend: API, dominio, aplicacao, infraestrutura e testes.
- frontend: aplicacao cliente React.
- docs: documentacao da Etapa 1.
- diagrams: diagramas em formato Mermaid.
